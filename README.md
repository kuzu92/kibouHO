# Role Assigner (同率順位対応・役職自動割り当てツール)

メンバーから提出された希望順位をもとに、各役職の定員に合わせて自動で均等に割り振るPythonツールです。

## 特徴
- **同率順位に対応**: 「A役もB役もどちらも1位」といった提出が可能です。
- **倍率ベースの最適化**: 希望が集中している（倍率が高い）役職から優先的にマッチングを確定させるため、全体の満足度が高くなります。
- **公平な抽選**: 同一の条件（同じ希望順位で同じ役職を狙う場合）がバッティングした際は、ランダムに抽選されます。

## 使い方

### 1. 準備
Python（3.x以降）が必要です。また、設定ファイルの読み込みに必要なライブラリをインストールしてください。
```bash
pip install pyyaml
```

### 2. 設定の編集
`config.yaml` をテキストエディタで開き、役職の定員とメンバーの希望順位を記入します。
- 数値が小さいほど優先度が高くなります（1 = 第一希望）。
- 同じ数値を書くと同率順位になります。

### 3. 実行
以下のコマンドを実行すると、ターミナルに結果が出力されます。
```bash
python assign.py
```
